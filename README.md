# Oops

CYCLE HACK TOKYO 2015で開発したiOSアプリです。
自転車で事故った時の振動をトリガーに登録した相手に電話をかけ、位置情報をサーバに送ることができる。
子供が事故にあった時に親にすぐ電話できるなどのシーンを想定している。
